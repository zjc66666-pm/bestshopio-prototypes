# BestShopio — Admin Prototypes

Live, clickable prototypes of the BestShopio merchant admin. One shared shell and
design system across every module; each PRD version adds to the same set.

## Structure
- `index.html` — hub: module map + changelog
- `assets/` — shared shell, theme and module manifest (single source of truth)
- `analytics/`, `orders/`, … — one folder per module

Synced from the private `bestshopio-proto` working repo. Served via GitHub Pages
(branch `main`, folder `/root`).
